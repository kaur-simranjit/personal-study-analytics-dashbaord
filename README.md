<h1>Personal Study Analytics Dashboard</h1>

<p>A web-based application where a student can track their study habits and see how those habits affect their productivity.</p>

<p>The application allows users to add study sessions, view all saved study sessions, and visualize their study patterns through a dashboard.</p>

<h3>Key Features</h3>
<ul style="list-style-type:disc">
  <li>📌 Add and store study sessions</li>
  <li>📋 View all saved sessions</li>
  <li>📊 Dashboard with key metrics:
    <ul>
        <li>Total study sessions</li>
        <li>Total hours studied</li>
        <li>Average productivity rating (1-5)</li>
        <li>Average break time</li>
    </ul>
  </li>
  <li>📈 Data visualisation:
    <ul>
        <li>Study hours over time</li>
        <li>Productivity over time</li>
        <li>Time spent by course</li>
        <li>Productivity by study method</li>
    </ul>
  </li>
  <li>🔍 Filter data over different time ranges<li>
</ul>
