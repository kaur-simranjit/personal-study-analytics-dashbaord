<h1>Personal Study Analytics Dashboard</h1>

<p>Personal Study Analytics Dashboard is a web-based application where a student can track their study habits and see how those habits affect their productivity.</p>
<p>The application allows the student to add study sessions by entering details such as date, course studied, number of hours studied, study method, break time, and productivity rating. It then allows to view all previously saved study sessions. With the provided data, it will calculate and visualize the study patterns through a dashboard.</p>
